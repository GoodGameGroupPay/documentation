**Group Game Group Pay**（简称**GGGPay**）允许开发者通过API来进行交易，主要关注通过RESTful API或SDK进行支付和提款。 所有GGGPay产品都有其大部分核心功能的 API。希望能通过GGGPay的全面API来重新构想您的业务。

## 环境

目前提供了两个环境来测试功能并确保无缝完成集成，有**沙盒**环境和**生产**环境。

### 沙盒 环境

登录系统 -> https://demo.gggpay.org/ <br />
API 接口服务器 -> https://testapi.gggpay.org <br />

### 生产 环境

登录系统 -> https://merchant.gggpay.org
API 接口服务器 -> https://api.gggpay.org

**开始之前**

前往[商户平台](https://merchant.gggpay.org) 注册成为商家

提交商家门户中显示的所有必需文件，并获得我们管理员的批准。 如果您需要更多帮助，请通过 <hello@gggpay.org> 联系我们。

SDK，请参阅 [SDK 部分](../quickstart/sdk)。
:::
